# ChinoC编译器

## C编译器的子集实现

### 注意⚠️：该编译器不是成品，以下已发现的bug列表
* 未回收内存，有着大量的内存泄露
* 不支持过于复杂的指针和数组的组合声明
* 段错误bug，不知道什么时候会触发
* switch的case不是常量也行

### 以下是feature
* 将C代码转换为HIR(High IR)

### 编译时注意事项
* 请在Linux环境下编译

### 其他注意事项
* 由于作者太鸽，不知道什么时候会来修复bug
* 代码有点混乱(基本无注释)，有空再整理代码
* 要读取c文件可以在main.c里修改yyin指针

## HIR的语法
* 将表达式转换为逆波兰表达式
* 将for循环转换为while循环
* 其他的和C一样
